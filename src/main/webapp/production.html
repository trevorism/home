Productionalized web service

What does a complete, productionalized web service look like?

  • Available over HTTPS
  • Unit Tested; Integration Tested; Regression Tested: Code Coverage (>60%)
  • Continuously integrated portable build
  • Deployable without outages
  • Logs can be used to debug transaction failures
  • Emits telemetry on its health; telemetry is monitored
  • Load Balanced for performance and failover
  • Computing resources added on demand
  • Hosting and endpoints discoverable
  • Artifacts discoverable and backed up
  • Access to sensitive data is secured by authentication and authorization
